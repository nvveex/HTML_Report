# 指标口径规则

## 核心汇总指标

对于具备时间序列特征的数据，尽量计算以下指标：
- `latest_period`：最近周期
- `latest_value`：最近周期值
- `peak_period`：峰值周期
- `peak_value`：峰值
- `total_value`：累计值
- `mean_value`：均值
- `volatility_ratio`：波动系数
- `top_category`：主导类别
- `top_category_share`：主导类别占比

## 定义说明

### 峰值

使用当前观测序列中的最大值。

### 最近周期

在日期标准化后，使用字典序最靠后的周期。

### 波动 / 离散程度

当存在重复周期观测时，优先使用变异系数：

`波动系数 = 标准差 / 均值`

参考解释：
- `< 0.25`：离散程度较低
- `0.25 - 0.60`：离散程度中等
- `> 0.60`：离散程度较高

只有在至少有 4 个周期且均值非 0 时，才适合讨论“离散程度”。

### 稳定性

稳定性 = 节奏连续 + 波动适中或较低。

如果波动很高，或存在长时间零值/空档，应判断为稳定性较弱。

### 结构集中度

对于分类分布，计算：

`主导类别占比 = 主导类别值 / 类别总量`

当占比 `>= 0.60` 时，可认为结构集中。

### 常模参照

本 skill 默认只允许做校内历史常模参照，例如：
- 最近周期与历史均值相比
- 开学阶段与常态周期相比
- 当前业务与该业务历史节奏相比

不允许暗示外部学校、行业、区域常模，除非用户明确提供。

## 问题识别启发式

满足以下任一条件时，可以列为问题候选：
- `volatility_ratio > 0.60`
- 在至少 6 个周期数据下，最近值低于峰值的 50%
- `top_category_share >= 0.70`
- 前期有活跃度，后续长时间接近 0
- 关键角色绑定率明显失衡

## 行动建议映射

建议动作应与问题类型对应：
- 低采用率：培训、流程嵌入、入口简化
- 角色失衡：按角色定向推动绑定与使用
- 结构过度集中：扩展应用场景、分散责任
- 节奏不稳定：建立固定检查点和周期机制
- 数据口径不稳：规范导出字段、命名和采集方式
